Политика конфиденциальности Somio

Дата вступления в силу: 18 января 2026 года

Команда Somio ("мы", "нас", "наш") серьезно относится к защите ваших персональных данных. Настоящая Политика конфиденциальности описывает, как мы собираем, используем, обрабатываем и защищаем информацию, предоставляемую вами при использовании нашего приложения и связанных сервисов.

1. Собираемая информация

В процессе использования приложения Somio мы можем собирать следующие категории данных:

1.1 Информация, предоставляемая пользователем

  • Профильные данные: имя, дата рождения, рост, пол
  • Записи веса: вес, дата и время измерения, примечания
  • Измерения тела: обхват груди, талии, бедер, шеи, бицепсов, предплечий, бедер, голеней
  • Фотографии: фотографии для визуального отслеживания прогресса (хранятся локально и в iCloud)
  • Цели: целевой вес, целевая дата, тип цели (похудение, набор веса, поддержание)
  • Настройки приложения: предпочтения пользователя, единицы измерения

1.2 Автоматически собираемые данные

  • Техническая информация: тип устройства, версия iOS, уникальные идентификаторы устройства
  • Данные использования: информация о том, как вы взаимодействуете с приложением (для улучшения функциональности)
  • Журналы и диагностика: данные об ошибках и производительности для улучшения работы приложения

2. Цели обработки данных

Собранная информация используется исключительно для следующих целей:

  • Предоставление и персонализация сервисов приложения
  • Отслеживание прогресса веса и измерений тела
  • Расчет показателей здоровья (ИМТ, процент жира, калории, макроэлементы)
  • Визуализация данных в виде графиков и диаграмм
  • Синхронизация данных между вашими устройствами через iCloud
  • Обеспечение безопасности и защита персональных данных
  • Анализ и улучшение функциональности приложения
  • Соблюдение юридических обязательств

3. Защита данных

Мы реализуем комплексные меры безопасности, включая:

  • Локальное хранение: все данные хранятся локально на вашем устройстве с использованием SwiftData
  • iCloud синхронизация: данные синхронизируются через защищенное соединение Apple iCloud с использованием CloudKit
  • Биометрическая защита: опциональная защита данных с помощью Face ID или Touch ID
  • Шифрование: все данные передаются и хранятся в зашифрованном виде
  • Ограниченный доступ: данные доступны только вам и синхронизируются только на ваших устройствах, связанных с одним Apple ID

4. Передача данных третьим сторонам

Мы можем передавать информацию в следующих случаях:

  • Apple iCloud: для синхронизации данных между вашими устройствами. Данные хранятся в вашем личном iCloud аккаунте и недоступны нам
  • Поставщикам услуг: только тем партнерам, которые помогают нам в работе приложения (аналитика, если используется), и только в анонимном виде
  • По юридическим требованиям: когда это необходимо для соблюдения законов или ответа на правомерные запросы
  • При корпоративных изменениях: в случае слияния, поглощения или продажи активов

Важно: Мы не продаем и не передаем ваши персональные данные третьим лицам для маркетинговых целей.

5. Международная передача данных

Учитывая использование Apple iCloud для синхронизации, ваши данные могут обрабатываться на серверах Apple, расположенных за пределами вашей страны проживания. Мы гарантируем, что такая передача осуществляется с соблюдением соответствующих мер защиты, установленных Apple и применимым законодательством.

6. Срок хранения данных

Мы храним ваши персональные данные только в течение срока, необходимого для достижения целей, указанных в настоящей Политике, или в соответствии с требованиями законодательства. Вы можете в любой момент удалить все данные через настройки приложения или удалив приложение с устройства.

7. Ваши права

Вы имеете право:

  • Запрашивать доступ к вашим персональным данным и их копию
  • Требовать исправления неточных или неполных данных
  • Запрашивать удаление ваших данных при отсутствии законных оснований для их хранения
  • Экспортировать ваши данные в формате JSON через функции приложения
  • Создавать резервные копии ваших данных
  • Ограничивать или возражать против обработки ваших данных в определенных случаях
  • Отзывать согласие на обработку, когда оно является правовым основанием для обработки

8. Разрешения приложения

Приложение Somio может запрашивать следующие разрешения:

  • Камера: для съемки фотографий записей веса (только с вашего разрешения)
  • Фото-библиотека: для выбора фотографий из вашей библиотеки (только с вашего разрешения)
  • Face ID / Touch ID: для защиты данных приложения (опционально, только с вашего разрешения)
  • Уведомления: для напоминаний о записи веса (опционально, только с вашего разрешения)

Все разрешения запрашиваются только при необходимости и могут быть отозваны в любой момент через настройки устройства.

9. Изменения в политике

Мы оставляем за собой право периодически обновлять настоящую Политику конфиденциальности. Все изменения вступают в силу с момента их публикации в приложении и на этой странице. Мы рекомендуем регулярно проверять эту страницу для ознакомления с актуальной версией.

The Somio team ("we", "us", "our") takes the protection of your personal data seriously. This Privacy Policy describes how we collect, use, process, and protect information you provide when using our application and related services.

In the course of using the Somio application, we may collect the following categories of data:

1.1 Information Provided by the User

  • Profile Data: name, date of birth, height, gender
  • Weight Entries: weight, date and time of measurement, notes
  • Body Measurements: chest, waist, hips, neck, biceps, forearms, thighs, calves circumference

1.2 Automatically Collected Data

  • Technical Information: device type, iOS version, unique device identifiers
  • Usage Data: information about how you interact with the application (to improve functionality)
  • Logs and Diagnostics: error and performance data to improve application performance

2. Purposes of Data Processing

The collected information is used exclusively for the following purposes:

  • Providing and personalizing application services
  • Tracking weight and body measurements progress
  • Calculating health indicators (BMI, body fat percentage, calories, macronutrients)
  • Visualizing data in charts and graphs
  • Synchronizing data between your devices via iCloud
  • Ensuring security and protecting personal data
  • Analyzing and improving application functionality
  • Complying with legal obligations

3. Data Protection

We implement comprehensive security measures, including:

  • Local Storage: all data is stored locally on your device using SwiftData
  • iCloud Sync: data is synchronized through Apple's secure iCloud connection using CloudKit

  • Limited Access: data is accessible only to you and synchronized only on your devices linked to the same Apple ID

We may share information in the following cases:

5. International Data Transfer

Given the use of Apple iCloud for synchronization, your data may be processed on Apple servers located outside your country of residence. We guarantee that such transfer is carried out in compliance with appropriate protection measures established by Apple and applicable legislation.

6. Data Retention Period

We store your personal data only for the period necessary to achieve the purposes specified in this Policy, or in accordance with legal requirements. You can delete all data at any time through the application settings or by deleting the application from your device.
